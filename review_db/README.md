# review_db

用于查看和删除 `outputs/` 目录下 3 个 SQLite 库中的记录：

- 手动测试评审库：`outputs/frontend_manual_test.sqlite3`
- 自动生成对话库：`outputs/generated_dialogues.sqlite3`
- 改写评审库：`outputs/frontend_rewrite_review.sqlite3`

## 三个库分别是什么

### 1. 手动测试评审库

- 文件：`outputs/frontend_manual_test.sqlite3`
- 表：`manual_test_reviews`
- 对应功能：前端测试模式里的手动会话评审落库
- 主键：`session_id`
- 时间列：`reviewed_at`

可查看的主要内容：

- `session_id`
- `scenario_id`
- `username`
- `status`
- `aborted_reason`
- `is_correct`
- `failed_flow_stage`
- `reviewer_notes`
- `persist_to_db`
- `started_at`
- `ended_at`
- `reviewed_at`
- `review_payload_json`

其中 `review_payload_json` 里主要还能看到：

- `transcript`
- `review`
- `call_start_time`
- `collected_slots`
- `session_config`

可删除的内容：

- 按 `session_id` 删除单条或多条记录
- 按时间范围删除
- 清空整表

### 2. 自动生成对话库

- 文件：`outputs/generated_dialogues.sqlite3`
- 表：`generated_dialogues`
- 对应功能：自动生成的对话结果落库
- 主键：`dialogue_id`
- 时间列：`generated_at`

可查看的主要内容：

- `dialogue_id`
- `scenario_id`
- `status`
- `rounds_used`
- `missing_slots_json`
- `collected_slots_json`
- `generated_at`
- `dialogue_payload_json`

其中 `dialogue_payload_json` 里通常还能看到：

- `transcript`
- `scenario`
- `missing_slots`
- `collected_slots`

可删除的内容：

- 按 `dialogue_id` 删除单条或多条记录
- 按时间范围删除
- 清空整表

### 3. 改写评审库

- 文件：`outputs/frontend_rewrite_review.sqlite3`
- 表：`rewrite_reviews`
- 对应功能：前端改写模式里“提交评审”后的记录落库
- 主键：`record_id`
- 时间列：`reviewed_at`

可查看的主要内容：

- `record_id`
- `username`
- `source`
- `reviewed_at`
- `review_payload_json`

其中 `review_payload_json` 里主要还能看到：

- `record`
- `record.conversations`
- `record.rewrited`
- `record.session_id`
- `record.auto_mode_id`

可删除的内容：

- 按 `record_id` 删除单条或多条记录
- 按时间范围删除
- 清空整表

## 如何查看

### 查看手动测试评审库

返回 JSON：

```bash
python -m review_db.fetch_reviews --db_path outputs/frontend_manual_test.sqlite3 --limit 20
```

按 `session_id` 查看：

```bash
python -m review_db.fetch_reviews --db_path outputs/frontend_manual_test.sqlite3 --session-id <session_id>
```

以可读文本输出：

```bash
python -m review_db.fetch_reviews --db_path outputs/frontend_manual_test.sqlite3 --format cli --limit 5
```

### 查看自动生成对话库

返回 JSON：

```bash
python -m review_db.fetch_generated --db_path outputs/generated_dialogues.sqlite3 --limit 20
```

按 `dialogue_id` 查看：

```bash
python -m review_db.fetch_generated --db_path outputs/generated_dialogues.sqlite3 --dialogue-id <dialogue_id>
```

按场景查看：

```bash
python -m review_db.fetch_generated --db_path outputs/generated_dialogues.sqlite3 --scenario-id <scenario_id>
```

以可读文本输出：

```bash
python -m review_db.fetch_generated --db_path outputs/generated_dialogues.sqlite3 --format cli --limit 5
```

### 查看改写评审库

返回 JSON：

```bash
python -m review_db.fetch_rewrite_reviews --db_path outputs/frontend_rewrite_review.sqlite3 --limit 20
```

按 `record_id` 查看：

```bash
python -m review_db.fetch_rewrite_reviews --db_path outputs/frontend_rewrite_review.sqlite3 --record-id <record_id>
```

按账号或来源筛选：

```bash
python -m review_db.fetch_rewrite_reviews --db_path outputs/frontend_rewrite_review.sqlite3 --username <username>
python -m review_db.fetch_rewrite_reviews --db_path outputs/frontend_rewrite_review.sqlite3 --source auto_mode
```

以可读文本输出：

```bash
python -m review_db.fetch_rewrite_reviews --db_path outputs/frontend_rewrite_review.sqlite3 --format cli --limit 5
```

## 如何删除

统一使用：

```bash
python review_db/delete_records.py --target <reviews|generated|rewrite> <strategy> [参数]
```

支持的删除策略：

- `by-id`
- `recent`
- `older-than`
- `before`
- `after`
- `all`

支持的全局开关：

- `--dry-run`：只看将删除多少条，不实际删除
- `--yes`：跳过确认

### 删除手动测试评审库

按 `session_id` 删除：

```bash
python review_db/delete_records.py --target reviews by-id <session_id>
```

删除最近 24 小时的评审：

```bash
python review_db/delete_records.py --target reviews recent --hours 24
```

预览 30 天前会删多少条：

```bash
python review_db/delete_records.py --target reviews --dry-run older-than --days 30
```

### 删除自动生成对话库

按 `dialogue_id` 删除：

```bash
python review_db/delete_records.py --target generated by-id <dialogue_id>
```

删除某个时间点之前的记录：

```bash
python review_db/delete_records.py --target generated before --time "2026-03-01T00:00:00"
```

### 删除改写评审库

按 `record_id` 删除：

```bash
python review_db/delete_records.py --target rewrite by-id <record_id>
```

删除最近 7 天的改写评审：

```bash
python review_db/delete_records.py --target rewrite recent --days 7
```

删除某个时间点之前的改写评审：

```bash
python review_db/delete_records.py --target rewrite before --time "2026-04-01 00:00:00"
```

### 清空整表

```bash
python review_db/delete_records.py --target reviews --yes all
python review_db/delete_records.py --target generated --yes all
python review_db/delete_records.py --target rewrite --yes all
```

## 建议

第一次执行任何删除操作时，先跑一遍 `--dry-run`。

例如：

```bash
python review_db/delete_records.py --target rewrite --dry-run recent --days 3
```
